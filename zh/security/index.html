安全与合规

我们知道信任是靠行动赢来的，不是靠口号喊出来的。下面是我们为赢得您信任而构建的每一项机制。

令牌化卡片、应用内 KYC、风险分级审批门、加密签名审计追踪以及隔离账户——每个账户均已内置，无需额外配置。

大多数金融科技产品只会告诉您您的资金是安全的。我们更愿意向您展示背后的机制。以下每一项安全控制默认开启，适用于每个账户，且不设付费门槛。如果您有具体问题，请通过我们的联系表单、Telegram @paype 或发送邮件至 hello@paype.cc 联系我们。

安全功能

  • ✓

    令牌化卡片

    您的真实卡号永远不会离开应用。Apple Pay 和 Google Pay 会为每笔交易生成设备专属令牌。即使商户被入侵，您的卡片仍然安全——只需发一个新令牌即可。您无需任何操作。

  • ✓

    应用内 KYC（了解您的客户）

    拍摄您的身份证件和自拍照即可开通账户。系统自动核验政府记录——大多数账户在几分钟内即可完成验证。无需分行，无需纸质文件。KYC 凭证可在 paype 生态系统中重复使用。

  • ✓

    风险分级审批门

    小额支付即时放行；大额支付需要二次确认。每笔交易根据金额、币种和方式划分为低、中、高三个风险等级。低风险交易自动放行。中高风险交易需要升级验证：根据阈值不同，可能需要 OTP、生物识别验证或双人审批。

  • ✓

    加密签名审计追踪

    每一笔操作都被记录并可验证——对您和您的会计师都是如此。每项改变状态的操作——发送、兑换、批准、冻结——都会被记录在防篡改的审计日志中。每条记录都带有可独立验证的加密签名。您可以追踪每一笔资金动向；审计人员无需向您索取任何资料即可验证完整的操作链。

  • ✓

    敏感操作多重审批

    在高价值或高风险操作中，任何人无法独自转移资金。敏感交易需要两个不同的授权人审批。第二方的审批将在审计日志中与第一方链接——任何一方都无法单独否认。

  • ✓

    隔离账户

    您的资金存放在与 paype 自有运营资金相隔离的账户中。paype 的业务财务无法触及您的余额。仅在严格管控下，资金才会在代理行池层面进行混存。

我的资金受 FDIC 保险保障吗？

不受保障。paype.cc 是一家货币服务企业（Money Services Business），而非银行。您的 paype 余额不属于银行存款，不受 FDIC 或任何其他政府存款保险计划的保障。

您的资金存放在与 paype 自有运营资金相隔离的独立账户中。这意味着 paype 的业务财务无法触及您的余额。但与银行存款不同的是，如果底层机构倒闭，没有政府担保机制兜底。

这就是 MSB 模式的取舍之道：更低的门槛、更少的手续费、更高的灵活性——但没有 FDIC 保障。请阅读我们的《水手须知》披露声明了解完整情况。

合规框架

paype.cc 作为一家在美国金融犯罪执法网络（FinCEN）注册的货币服务企业（MSB）运营，注册依据为 31 CFR §1022.380。您可以在 fincen.gov/msb-registrant-search 验证我们的注册信息。

合规计划包括：

  • BSA/AML 计划：遵守《银行保密法》/反洗钱规定，配备基于风险的政策、流程和内部控制措施。
  • 制裁筛查：所有交易实时比对美国及国际制裁名单进行筛查。
  • 旅行规则合规：对于超过适用阈值的加密货币交易，按规定收集并传输旅行规则信息。
  • 可疑活动报告（SAR）：按规定向 FinCEN 提交可疑活动报告，每份报告均需多重审批。
  • 独立审计：合规计划定期接受独立审查。

账户控制

您自主掌控账户安全：

  • 生物识别登录：使用 Face ID 或指纹解锁应用。
  • 交易通知：每笔资金进出实时推送通知。
  • 冻结卡片：卡片遗失时一键锁定——找回后即可解冻。
  • 消费限额：设置每日、每笔交易及每类别的消费上限。
  • 基于位置的规则：仅当您的手机在附近时才允许交易。
  • 设备管理：查看并撤销所有登录您账户的设备。
